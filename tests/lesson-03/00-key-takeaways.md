Các lệnh git:
    -  git commit —amend -m “<tên commit mới> “	: Câu lệnh thay đổi commit message
    - git restore —staged <file name>:	loại bỏ một file ra khỏi Staging Area (vùng chờ)
    - git reset - - soft HEAD ~1: đưa nội dung commit cuối cùng về staging
    - git reset HEAD~1: đưa nội dung commit cuối cùng về working directoty
    - git branch < tên nhánh>: 	tạo nhánh
    - git checkout <tên nhánh>: 	checkout qua nhánh đó
    - git checkout -b < tên nhánh>: 	tạo nhánh + chuyển qua nhánh đó luôn
    - git checkout main	quay trở lại main
    - git branch: 	xem tất cả các nhánh , xem mình đang đứng nhánh nào
    - git branch -D <tên nhánh>:	Xóa nhánh dưới máy cá nhân
    - git push origin -D <tên nhánh>:	xóa nhánh trên online
